# Cursed
код тулла для пробива.

Для запуска откройте Kills.py.

Почему файл называется kills а тулл cursed?

Тулл ещё в разработке и мы только что поменяли название.

Работает пока-что только на windows

В будущем будет порт на Linux и Termux

Тулл имеет такие фукции как:
1. Поиск по номеру
2. SMS BOMBER
3. Поиск по IP
4. DDoS


Как запустить?

Windows:

Скачиваем репозитрий в .zip формате.

Разархивируем и запускаем modules.bat, далее запускаем loader.py

Всё просто.

Желаем всем удачного пользования, все права защищены.
